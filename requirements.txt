# forge-narrator has NO runtime Python dependencies.
#
# - ElevenLabs is called over HTTP using the standard library (urllib) — no SDK.
# - ffmpeg + ffprobe are HARD dependencies but are NOT pip packages — install via
#   Homebrew:  brew install ffmpeg   (already present on the Air, v8.x).
#
# Install the package itself (editable) into a Python 3.11 venv:
#   python3.11 -m venv .venv && .venv/bin/pip install -e .
#
# For the test suite:  .venv/bin/pip install -e '.[dev]'   (pytest)
#
# The local web console (`forge-narrator serve`) needs FastAPI + uvicorn:
#   .venv/bin/pip install -e '.[web]'
#     fastapi, uvicorn[standard], python-multipart
